# &lt;console-js&gt;

> JavaScript console enhancements

## Install

Install the component using [Bower](http://bower.io/):

```sh
$ bower install consolejs --save
```

Or [download as ZIP](https://github.com/jordifebrer/console-js/archive/master.zip).
